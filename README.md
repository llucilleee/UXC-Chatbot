# UXC-AI-Chatbot

# AI Chatbot Check‑in System

## Project Overview

The AI Chatbot Check‑in System is designed to enhance the User Experience Centre (UXC) by collecting structured feedback from trainers and colleagues through an automated conversational interface. The system aims to streamline feedback collection, improve data accuracy, and generate actionable insights for continuous improvement.

This project will support internal stakeholders by providing a smarter and more efficient way to collect, analyze, and visualize feedback data.

# System Flow

## Check‑in Flow

1. User opens system
2. AI Chatbot welcomes user
3. User selects check‑in option
4. Chatbot asks user details
5. Chatbot asks session details
6. Chatbot collects feedback
7. Chatbot confirms submission
8. Data stored in database
9. Insights generated
10. Dashboard updated
